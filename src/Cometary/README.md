# Cometary

Referencing this project will cause the following packages to be referenced as well:
- [Cometary.Generation](../Cometary.Generation)
- [Cometary.Rewriting](../Cometary.Rewriting)
- [Cometary.Scripting](../Cometary.Scripting)
